# AzureLocal

Infrastructure, automation, and tooling for **Azure Local** (formerly Azure Stack HCI) deployments — designed for distributed enterprise environments at the edge and in on-premises datacenters.

## Repositories

| Repository | Description |
|------------|-------------|
| [azurelocal.github.io](https://github.com/AzureLocal/azurelocal.github.io) | Central documentation site at [azurelocal.cloud](https://azurelocal.cloud) |
| [azurelocal-ranger](https://github.com/AzureLocal/azurelocal-ranger) | Discovery and readiness assessment tool |
| [azurelocal-avd](https://github.com/AzureLocal/azurelocal-avd) | Azure Virtual Desktop on Azure Local |
| [azurelocal-sofs-fslogix](https://github.com/AzureLocal/azurelocal-sofs-fslogix) | Scale-out File Server + FSLogix profile containers |
| [azurelocal-loadtools](https://github.com/AzureLocal/azurelocal-loadtools) | Load testing framework for Azure Local workloads |
| [azurelocal-vm-conversion-toolkit](https://github.com/AzureLocal/azurelocal-vm-conversion-toolkit) | VM migration and conversion tools |
| [azurelocal-toolkit](https://github.com/AzureLocal/azurelocal-toolkit) | General-purpose Azure Local automation toolkit |
| [azurelocal-nutanix-migration](https://github.com/AzureLocal/azurelocal-nutanix-migration) | Nutanix to Azure Local migration guidance |
| [azurelocal-training](https://github.com/AzureLocal/azurelocal-training) | Training materials and lab guides |
| [azurelocal-copilot](https://github.com/AzureLocal/azurelocal-copilot) | GitHub Copilot customizations for Azure Local development |

## Standards

All repositories follow the [AzureLocal Standards](https://azurelocal.cloud/standards/) — covering naming conventions, repository structure, variable handling, and contribution guidelines.
